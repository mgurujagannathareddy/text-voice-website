## âš™ï¸ Functional Modules

### 1. Text Input Module
- Allows users to enter text
- Uses a `<textarea>` element
- JavaScript reads the input dynamically

---

### 2. Speech Rate Control
- Allows users to control speech speed
- Implemented using a slider (`input type="range"`)
- Uses `speech.rate` property

---

### 3. Pitch Control
- Allows users to adjust voice pitch
- Makes speech sound more natural
- Uses `speech.pitch` property

---

### 4. Voice Selection
- Displays available system voices
- Supports different accents and languages
- Uses `speechSynthesis.getVoices()`

---

### 5. Text-to-Speech Engine
- Converts text into audible speech
- Uses `SpeechSynthesisUtterance`
- Speech output plays through system speakers
